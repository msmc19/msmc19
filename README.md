### 🔍 Fraud Detection & Risk Analytics Pipeline
End-to-end pipeline processing **250K+ digital wallet transactions** with five engineered anomaly indicators — **value, velocity, geolocation, category, and payment method** — feeding Tableau visualizations that drive automated alerting workflows.
> `Python` · `Pandas` · `Tableau`

### 🧾 Modular Tax Return Management System
Normalized **PostgreSQL** schema with enforced one-return-per-client integrity, paired with an OOP Python backend featuring **connection pooling, parameterized queries, and full CRUD** — supporting multi-role user management across CPAs, clients, and assistants.
> `Python` · `PostgreSQL` · `OOP`

### 🌐 ISP Customer Tracking & Billing System
Normalized SQL schema spanning **Customers, Locations, Services, Equipment, and Billing** entities, accessed through a Python CLI with a clean database abstraction layer enabling full CRUD management and automated overdue-payment reporting.
